All Mac users, if you want to be in-sync with my environment, you may install same Ruby version from Terminal by running following command:
curl -sSL https://get.rvm.io | bash -s stable --ruby=ruby-2.3.0
if you do not get any errors, you will need to run following command  thereafter:
rvm --default use 2.3.0
then, 
ruby -v 
will show if you have correct Ruby version installed
